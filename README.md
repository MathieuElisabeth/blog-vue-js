# Blog

Mini blog build with Vue JS & Firebase

## Project setup
```
npm install
```
Put your firebase config in ```firebase/config.js```, to connect at your database

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
